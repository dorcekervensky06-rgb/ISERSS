# ISERSS - TC PM

Institut Supérieur d’Études et de Recherche en Sciences Sociales  
Tronc Commun PM - Plateforme de gestion de cours

##  Description

Aplikasyon ISERSS TC PM lan se yon sistèm jesyon kou pou:
- Etidyan
- Pwofesè
- Administrasyon

Li pèmèt:
- Login ak yon sèl kòd
- Ajoute epi gade kou
- Wè etidyan ki konekte
- Sèvi ak Firebase realtime
